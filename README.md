A função desta classe é remover acentos e normalizar um link para web

Exemplo de entrada:

$entrada          = "O Universo é muito estranho?";

$saida            = "o-universo-e-muito-estranho";

$saidaAlternativa = "o_universo_e_muito_estranho";
